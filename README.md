# opzt学習記録アプリ

## 概要
製作、壁面、遊びなど、保育のアイデアを共有できるサイトを作成しました。

## 使い方
自分の作成した製作物や遊びのアイデアなどを写真付きで投稿したり、カテゴリやキーワードから作品検索することができます。

気に入ったアイデアがあればいいねすることができ、お気に入り一覧ページからまとめて確認できます。
